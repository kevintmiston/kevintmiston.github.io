TEKNIK KOMPUTER

Sistem Pendeteksi Kebocoran Gas dan Kebakaran Menggunakan Arduino UNO dengan Notifikasi telegram BOT

Penelitian ini bertujuan untuk menghasilkan sebuah alat pendeteksi kebocoran tabung gas LPG dengan menggunakan sensor MQ-2 sebagai sensor gas, dan ethernet shield sebagai modul pada mikrokontroler arduino uno untuk menghubungkan arduino dengan jaringan internet.

Sistem Keamanan Sepeda Motor menggunakan Raspberry Pi3 berbasis IOT

Sistem keamanan ini dibuat dengan sistem IOT yang mana sistem terintegrasi dengan internet. Kegunaannya untuk mengurangi terjadinya pencurian sepeda motor dan bisa mengetahui siapa saja yang memakai motor tersebut.

Mobile Remote Menggunakan Arduino UNO dengan Pengendali Via SmartPhone

Projek ini dibuat dengan menggunakan beberapa komponen dan untuk mikrokontrolernya sendiri yaitu Arduino UNO dan untuk mengendalikan menggunakan Aplikasi di SmartPhone dengan koneksi Bluetooth.
